Realistic Planet Visuals — Procedural Preview

Approximation of what the Three.js GLSL shaders will produce. All generated from data, no textures.

Ocean Planet

Rocky / Terrestrial
Mars/Moon-class
Cratered rust-red surface, volcanic highlands, dust storms. Lava cracks appear if temp exceeds 1000K.
In Three.js: Each planet is a THREE.Sphere with a custom ShaderMaterial using Simplex noise in GLSL to generate terrain, bands, and clouds procedurally — seeded from the planet's name so each one is unique but consistent. A second transparent sphere layer handles clouds and atmosphere glow. Drag-to-rotate and zoom included.
